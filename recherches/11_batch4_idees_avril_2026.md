### ID 220 — Mini-Cours en Ligne "IA pour une Profession"

**Concept :** Cours Gumroad/Teachable ($47-97) qui enseigne à une profession non-technique comment utiliser l'IA dans son quotidien : comptables (automatiser les rapports), RH (rédiger JD et processus de recrutement), juristes (résumer des contrats), assistants médicaux (notes de consultation). L'IA est omniprésente mais 80% des professionnels ne savent pas comment l'utiliser concrètement.

**Données :** Recherches sur l'IA pour les professions non-tech = ×2 en 2026. Un cours créateur a généré $150K en 6 mois avec des outils IA pour scripter les leçons. Gumroad Writing & Publishing = revenu médian par produit le plus élevé de toutes catégories.

**Anti-duplication :** Différent de LinkedIn Ghostwriting (id 175) et de AI Workflow Automation pour Avocats (id 193). Ce sont des cours autonomes asynchrones, pas des services de prestation.

---

### ID 221 — Ebook Guide Pratique IA pour Solopreneur (Gumroad)

**Concept :** Guide PDF actionnable ($17-47) : "Build a One-Person AI Agency in 30 Days", "100 Prompts Claude pour Freelances", "AI Stack Complet du Solopreneur 2026". Format ultra-actionnable avec checklists, templates et exemples.

**Données :** Gumroad Writing & Publishing = catégorie avec le plus haut revenu médian/produit (226 produits, $15 750 chacun, médiane 41 ventes). Pas de concurrent direct sur les guides IA + solopreneur spécifiquement.

### ID 222 — Etsy Planner Stickers Sheets (Bullet Journal)

**Concept :** Feuilles de stickers imprimables pour planners, bullet journals, agendas. Designs par thème : saisons, professions, self-care, habitudes, tâches ménagères. Le client achète régulièrement de nouveaux designs.

**Données :** Top sticker shops = centaines de feuilles/mois à $3-8 chacune. Taux de retour client = le plus élevé de Etsy. Achat répété = revenus stables. "Loyal sticker customers buy a new sheet every week."

### ID 223 — Etsy Lightroom Presets & Photo Filters

**Concept :** Presets Lightroom pour photographes, créateurs de contenu Instagram/TikTok, blogueurs de voyages. Pack à $10-25. Catégories : Golden Hour, Dark & Moody, Clean & Airy, Film Aesthetic.

**Données :** Lightroom presets = top 5 Gumroad 2026 dans la catégorie créative. Des créateurs solo rapportent $3K-8K/mois sur Etsy avec des collections de presets. Différent de SVG Cut Files (id 218) : audience et format différents.

### ID 224 — AI Property Description Generator (Agents Immobiliers)

**Concept :** SaaS qui génère des descriptions de biens immobiliers à partir de photos + caractéristiques (surface, pièces, localisation). Output : annonce pour SeLoger/Leboncoin, post Instagram, flyer PDF. 87%+ des agents immobiliers utilisent déjà l'IA.

**Données :** 97% des brokerage leaders confirment leurs agents utilisent l'IA (Jan 2026). Ticket $29-79/mois. Distribution via réseaux d'agents (associations, salons). Différent de Meeting Notes Juridique/Médical (id 207) : immobilier, pas meetings.

### ID 226 — AI Product Photo Generator pour E-commerçants

**Concept :** SaaS qui transforme une photo produit simple (fond neutre) en lifestyle shots professionnels avec différents décors et ambiances. Pour Shopify/Amazon sellers sans budget photo studio. Concurrent : Booth.ai ($20+/mois), RemoveBG. Mais le marché est large.

**Données :** Les e-commerçants dépensent $500-2K par shooting photo. Un outil à $19-49/mois = ROI immédiat. Marché : 4M boutiques Shopify + millions de sellers Amazon.

**Modèle :** $19-49/mois par boutique. 500 clients = $10K-$25K/mois.

### ID 227 — MCP Server Commercial pour une Profession Spécialisée

**Concept :** Créer et commercialiser un MCP (Model Context Protocol) server qui connecte des sources de données métier aux LLMs. Exemple : MCP server pour agents immobiliers (données SeLoger, DVF), pour juristes (Légifrance), pour comptables (API compta). Distribution GitHub + abonnement pour usage commercial.

**Données :** Greg Isenberg (2026) : "Building a MCP server in 2026 is like building for mobile in 2010 — early movers will own AI-native distribution channels." Anthropic MCP = standard en train de s'imposer. Première vague d'industrialisation des MCP servers.

**Modèle :** Gratuit (open source) → abonnement commercial $29-79/mois pour usage professionnel + support.

### ID 228 — Newsletter Niche B2B IA Premium (RH, Finance, Droit)

**Concept :** Newsletter hebdomadaire premium sur l'utilisation de l'IA dans une profession spécifique : "AI pour les DRH" ($12/mois), "IA & Comptabilité" ($12/mois), "IA pour les Avocats" ($15/mois). Format digest : 1 cas pratique + 2 outils + 1 prompt du mois.

**Données :** Newsletter premium = $5K-$52K/mois documenté pour les communautés early-stage sur Beehiiv/Substack. 1 000 abonnés à $12/mois = $12K/mois. Niche B2B = abonnés payants plus faciles à convertir.

**Anti-duplication :** Différent de id 13 (Newsletter Beehiiv généraliste) et id 70 (autre newsletter généraliste). Ce sont des newsletters B2B hyper-niche ciblant des professionnels qui ont un besoin urgent de comprendre l'IA dans leur métier.

### ID 235 — Programmatic SEO Tool pour Niches Locales

**Concept :** SaaS qui génère automatiquement des centaines de pages SEO pour des recherches locales ("meilleur plombier Paris 15", "avocat divorce Lyon 3"). Pour créateurs de sites de niche locale, agences marketing local, affiliateurs géo-cibles. Base de données + templates automatisés = PGSEO à grande échelle.

**Données :** Greg Isenberg cite le "programmatic SEO" comme une des stratégies de croissance clés en 2026. Les sites de niche locale avec pSEO génèrent du trafic organique massif. Marché : agences locales, sites d'annuaires, affiliateurs locaux.

**Modèle :** $49-149/mois. 100 clients = $5K-15K/mois.

### ID 242 — YouTube Faceless : Histoire de la Tech (Apple, Google, Startups)

**Concept :** Chaîne documentaire YouTube sur l'histoire des grandes entreprises tech : comment Apple a failli mourir en 1997, la vraie histoire de Google, l'ascension et la chute de startups emblématiques. Format 15-30 min. RPM fort ($8-15, audience tech/business).

**Données :** Niche "tech history" = RPM $8-15. Chaîne Company Man (3M abonnés) valide la demande. Contenu 100% scriptable par IA. Evergreen total (l'histoire d'Apple en 1997 reste vraie en 2030).

**Différent de YouTube Business Case Studies (id 179)** : id 179 = comment une entreprise a grandi. Id 242 = histoire dramatique + narratif journalistique. Format et angle éditorial différents.

### ID 243 — AI Chatbot Builder White-Label pour Agences Digitales

**Concept :** Plateforme white-label permettant aux agences web/marketing de revendre des chatbots IA à leurs clients (restaurants, boutiques, hôtels). Les agences payent $99-299/mois, revendent $200-500/mois à leurs clients.

**Données :** Marché du chatbot B2B en forte croissance. La différence clé : white-label = les agences revendent sous leur marque. Différent de id 43 (chatbots PME directement) : distribution B2B2C via agences, économies d'échelle.

### ID 244 — Cours Vidéo en Cohorte sur Claude/IA pour PME

**Concept :** Cours en direct (Zoom + replay) en cohorte mensuelle sur l'utilisation de Claude et des outils IA pour les TPE/PME. Durée : 4 sessions de 1h30. Prix : $197-497/cohorte. Certifications incluses.

**Données :** Format cohorte = urgence + communauté + présentiel online = forte valeur perçue vs cours asynchrone. $197 × 30 participants = $6K par cohorte. 12 cohortes/an = $72K. Top créateurs de cours font 100K+/an seuls avec l'IA pour produire les leçons.

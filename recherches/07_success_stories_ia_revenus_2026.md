## Resume executif

Le marche micro-SaaS IA est en pleine explosion : $15.7B en 2024, croissance ~30%/an, projete a $59.6B en 2030. Les produits IA-native atteignent $5K MRR environ 2x plus vite que les equivalents non-IA. La mediane profitable est a ~$4.2K MRR, mais les meilleurs solopreneurs depassent $50K-$300K/mois.

**Statistiques cles :**
- 70% des micro-SaaS gagnent < $1K MRR
- 18% entre $1K-$5K MRR
- Mediane profitable : ~$4.2K MRR
- Temps median pour $1M ARR : 2 ans 9 mois
- Marges operationnelles solo-founder IA : 60-80%
- Cout stack complet solopreneur : $3K-$12K/an

## Patterns de Succes Identifies

### Ce qui marche :
- **Speed to market** : TypingMind lance 5j apres l'API OpenAI, HeadshotPro construit en 30j
- **Niche > generique** : Subscribr (YouTube only), Koala (SEO only), Jenni (academique only)
- **Distribution virale** : Chatbase via TikTok, Sleek via X, HeadshotPro via SEO + affiliation
- **Zero API cost** : TypingMind fait payer l'utilisateur pour sa propre cle API = 100% marge
- **Chrome Extensions** : Distribution gratuite, cout quasi nul, audience captive
- **Pre-vente** : Subscribr a fait $20K avant d'ecrire une ligne de code

### Stacks dominants solo-founder :
- **Backend** : Laravel (PHP), Next.js, ou PHP vanilla
- **Hosting** : DigitalOcean, Vercel, Railway
- **IA** : OpenAI API, Anthropic API, Replicate (pour images)
- **Paiement** : Stripe, LemonSqueezy
- **DB** : Supabase, SQLite, PostgreSQL
- **Vector DB** : Pinecone, Supabase pgvector

### Pricing gagnant :
- Consumer : $9-$49/mois
- Prosumer : $29-$99/mois
- B2B : $99-$399/mois
- Licence unique (extensions) : $29-$79
- Pay-per-use : $5-$69 par session

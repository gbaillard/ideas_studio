# Recherche 08 -- Micro-Niches Etsy/Amazon/Gumroad SOUS-EXPLOITEES 2026

> Date : 2026-04-10 | Plateformes : Etsy, Amazon KDP, Gumroad, Creative Fabrica
> Objectif : Identifier 20+ micro-niches digitales sous-exploitees pour solopreneur IA
> Exclusions : langues/language learning, sport, nutrition, fitness, meditation
> Statut : 22 micro-niches identifiees avec fiches detaillees

---

## Resume executif

Cette recherche identifie des **micro-niches sous-exploitees** en 2026, en evitant les categories deja saturees (templates generiques, coloring books basiques, clipart standard, SVG generiques, stickers basiques, wall art generique).

**Mega-tendances cles 2026 impactant les micro-niches :**

1. **Human-AI Hybrid** -- Le contenu 100% IA est rejete. L'acheteur veut IA + retouche + style personnel.
2. **Editable > Static** -- Les PDFs statiques meurent. Canva/Corjl/Fillable PDF dominent.
3. **Hyper-specifique > Generique** -- "Budget planner pour freelance photographe" bat "budget planner".
4. **Medium-content > Low-content** -- Les livres KDP a contenu moyen (50-150 pages guidees) ecrasent les notebooks vides.
5. **Texture & Imperfection** -- Tendance Creative Market : "Slowly Knit", "Rococo Romance", hand-drawn revival.
6. **Bundles systemes** -- Les acheteurs veulent des systemes complets, pas des fichiers isoles.
7. **Low-ink printables** -- Economie d'encre = argument de vente reel en 2026.

### NICHE 1 -- Kits Escape Room Printables Thematiques

| Critere | Detail |
|---------|--------|
| **Plateforme** | Etsy (principal) + Gumroad |
| **Concurrence actuelle** | Faible -- Quelques centaines de listings Etsy, mais la majorite sont generiques. Les themes ultra-specifiques (decennies 70s/80s, dark academia, cottagecore, Wicked) ont < 50 listings chacun |
| **Difficulte** | 3/5 -- Necessite conception de puzzles logiques + design graphique |
| **Investissement** | 15-40 EUR/mois (Canva Pro + Midjourney) |
| **Revenu estime** | 800-4 000 EUR/mois (catalogue 10-20 kits) |
| **Prix moyen** | 8-15 EUR par kit, 25-40 EUR pour bundles |
| **Pourquoi sous-exploite** | Les escape rooms printables necessitent plus de travail que des templates classiques, ce qui decourage la majorite. Les themes micro-niches (film/serie specifique, epoque historique, profession) sont quasi vides. |
| **Outils IA** | ChatGPT/Claude (creation puzzles logiques, enigmes, indices), Midjourney (illustrations thematiques, cartes, accessoires visuels), Canva (mise en page, design cartes) |

**Process :**
1. Identifier un theme trending avec peu de resultats Etsy (ex: "1970s escape room printable", "dark academia escape room", "cottagecore mystery game")
2. Creer 8-12 puzzles varies avec ChatGPT/Claude : codes, cryptogrammes, labyrinthes, puzzles logiques, enigmes texte
3. Designer les visuels avec Midjourney (accessoires thematiques, arriere-plans, cartes d'indices)
4. Assembler dans Canva : cartes puzzle, feuille reponses, guide hote, decorations imprimables
5. Tester le kit (gameplay 30-60 min), ajuster difficulte
6. Proposer 3 niveaux : enfants (7-12), ados, adultes

**Sous-niches a explorer :**
- Escape rooms par decennie (60s, 70s, 80s, 90s)
- Escape rooms saisonniers (Halloween, Noel, Saint-Valentin, Paques)
- Escape rooms par univers (steampunk, dark academia, cottagecore, sci-fi retro)
- Escape rooms educatifs (sciences, histoire, geographie)
- Escape rooms pour "date night" (couple focus)

### ID 195 — Shopify FAQ Auto-Generator IA

**Concept :** SaaS qui se connecte à une boutique Shopify, scanne les tickets support existants + les descriptions produits, et génère automatiquement une page FAQ SEO-optimisée avec un widget embeddable. Résout un problème quotidien : les marchands reçoivent les mêmes questions et n'ont pas le temps de rédiger une FAQ structurée.

**Données :** Shopify héberge 4+ millions de boutiques actives. Le support client est le pain point #1 pour les petits commerçants. Les apps Shopify permettent la distribution sans marketing payant (App Store natif). Revenus dev Shopify : $93K/an en moyenne, top 25% = $167K.

**Anti-duplication :** Aucune idée Shopify-spécifique dans le catalogue (id 43 = chatbots PME généralistes, pas Shopify-natif).

**Build :** Shopify Partner API + OpenAI/Claude pour la génération → Bubble.io ou Next.js. Délai : 3-6 semaines.

**Modèle :** $29-49/mois par boutique. 200 boutiques = $6K-$10K/mois.

### ID 196 — Shopify Store SEO Audit Tool

**Concept :** Outil d'audit SEO spécialisé Shopify : analyse vitesse, schéma structuré, maillage interne, opportunités de mots-clés produits. Génère un rapport actionnable avec scores et recommandations priorisées.

**Différent de SEO generalist tools :** Pas de Semrush/Ahrefs, mais un outil centré uniquement sur les patterns SEO Shopify (collections, variants, pagination, etc.). Très niche, très lisible pour les marchands.

**Modèle :** $29-49/mois. Distribution via Shopify App Store.

### ID 197 — Shopify Product Image Alt-Text Generator IA

**Concept :** Scanne toutes les images produits d'une boutique Shopify et génère automatiquement des alt-texts SEO descriptifs et pertinents. Tâche ultra-répétitive que les marchands font mal (ou pas du tout).

**Simplicité :** Le scope est minimal = facile à builder en 2-3 semaines. Un seul usage clair = conversion App Store élevée.

**Modèle :** $19-39/mois. Très basse complexité = haute marge.

### ID 198 — KDP Bold & Easy Coloring Books (Seniors/Débutants)

**Concept :** Livres de coloriage avec contours épais (3-5px), formes simples et larges, zéro détail intrusif. Cible : seniors, personnes avec dextérité réduite, débutants, stress-relief casual. Format qui a explosé fin 2024 et domine 40% des bestsellers coloring KDP en 2026.

**Données :** "Bold and Easy" = tendance #1 coloring KDP 2026. Revenus série 5 livres = $3K-$10K/mois. BSR fort, faible retour. RPM royalties 60% à $9.99+.

**Anti-duplication :** Différent de l'id 6 (coloring books généralistes) : format spécifique, audience et stratégie tarifaire différentes. Le Bold & Easy cible une démographie distincte et génère des royalties plus élevées que les coloring books intricats.

**Sous-niches Bold & Easy :** Mandalas simples, Animaux outline, Scènes de la nature, Portraits géométriques, Patterns floraux.

### ID 199 — KDP Spot the Difference Books

**Concept :** Livres "trouve les différences" avec illustrations variées (villages médiévaux, marchés, scènes de nature, personnages). Niche peu exploitée car perçue comme "difficile à créer" — ce qui réduit la concurrence.

**Données :** KDPEasy classe les spot-the-difference books dans les top niches 2026 à "haute difficulté de création = faible concurrence". Cible : enfants, adultes, seniors. BSR prouvé.

**Build :** Midjourney/DALL-E pour générer les scènes → Photoshop/Canva pour introduire les différences subtiles. Plus de travail mais barrière à l'entrée plus haute = moins de saturation.

### ID 200 — KDP Guest Books Airbnb/VRBO Personnalisés

**Concept :** Livres d'or pour hôtes Airbnb/VRBO avec design thématique (maison de plage, chalet montagne, appartement urbain, mas provençal). Les hôtes veulent créer une "expérience" — un beau livre d'or y contribue.

**Anti-duplication :** Distinct de travel planners (id 116) : c'est un outil pour les hôtes, pas les voyageurs.

### ID 201 — KDP Medical Trackers (Diabète, Tension Artérielle)

**Concept :** Journaux de suivi médical quotidien : glycémie (diabétiques), tension artérielle (hypertendus), dosage médicaments, résultats de lab. Achat par "nécessité" et non par envie — taux de conversion très différent des planners lifestyle.

**Données :** 537M diabétiques dans le monde, 1.28 milliard d'hypertendus. Ces trackers ont des BSR stables toute l'année (pas de saisonnalité). Genre KDP peu exploré car les créateurs se concentrent sur les planners lifestyle.

**Anti-duplication :** Id 115 (logbooks santé) est un journal de santé généraliste lifestyle. Les trackers médicaux sont des outils de suivi clinique quotidien pour des conditions spécifiques — achats récurrents, motivation d'achat différente.

### ID 205 — AI Dunning Tool (Recouvrement de Paiements Échoués)

**Concept :** SaaS qui automatise le recouvrement des paiements échoués pour les plateformes d'abonnement : relances email intelligentes, retry logic (meilleure heure de retry), mise à jour CB expirées, notifications Slack/SMS. Cible : SaaS B2B, créateurs avec abonnements, e-commerce subscription.

**Données :** Les entreprises SaaS perdent en moyenne 9% de leur MRR à cause des paiements échoués (churn involontaire). Pour un SaaS à $10K MRR = $900/mois récupérables. ROI immédiat pour le client. Concurrent : Churnbuster ($99-399/mois), Gravy ($1K+/mois — trop cher pour petits SaaS).

**Modèle :** $49-199/mois. Niche : petits SaaS et créateurs avec subscriptions Stripe. 100 clients = $5K-$20K/mois.

### ID 214 — YouTube Faceless : True Crime Cold Cases

**Concept :** Chaîne centrée exclusivement sur les cold cases peu médiatisés (pas les célèbres — déjà saturés). Format : narration IA + voix over + images stock + documents d'archives. RPM parmi les plus élevés sur YouTube.

**Données :** True crime = RPM $12-18 (top 3 niches RPM sur YouTube). Sous-niche cold cases peu connus = moins de concurrence directe. Une chaîne à 100K abonnés dans cette niche peut générer $4K-$12K/mois.

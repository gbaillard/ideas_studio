# Cartes de Tarot sur Etsy : $252K de profit/an (etude de cas Dark Forest)

## Source YouTube
- Titre : How This Etsy TAROT CARD Seller Makes $252,000 PROFIT Per Year
- URL : https://www.youtube.com/watch?v=326zzvg9ffY
- Chaine : Brand Creators (Scott)
- Vues : 29K
- Duree : 14 min

## Process etape par etape

### Etape 1 : Recherche de niche avec Everbee
- Taper "tarot cards" sur Etsy
- Utiliser **Everbee** (extension Chrome) pour voir les estimations de recherches mensuelles
- Donnees trouvees : 3 217 recherches/mois pour "tarot cards", 2 330 pour "tarot card holder/box"
- Cliquer sur "Product Analytics" et trier par chiffre d'affaires mensuel
- Identifier les boutiques qui dominent (plusieurs listings dans le top)

### Etape 2 : Analyse de la boutique modele (Dark Forest Tarot Cards)
- 54 827 ventes totales
- Listing principale : active depuis 27 mois, 967 ventes, **$45 982/mois** estime
- 34 550 mises en favori = demande massive
- La boutique fait -70% de promo permanente
- CA reel estime : $140K brut x 30% (apres promo) = **$42 000/mois reel**
- 4 listings dans le top des meilleures ventes

### Etape 3 : Calcul des couts avec Print on Demand
- Fournisseur trouve : **Kin Printing** (impression a la demande de cartes)
- Commande de 500 jeux = $2 000 soit **$4/jeu** (78 cartes par jeu)
- Astuce recherche : taper "print on demand [nom du produit]" sur Google
- Cout total par commande : $4 (cartes) + $3 (divers/livraison) = **$7/unite**

### Etape 4 : Calcul de la marge avec Everbee Calculator
- Prix affiche : variable (avec promo -70%)
- Frais de port client : $8,80
- Cout article : $7
- Frais Etsy inclus dans le calcul
- **Marge nette : 50,65%** = $17,70 de profit par vente
- $42 000/mois x 50% = **$21 000 de profit net/mois**
- x12 = **$252 000/an de profit**

### Etape 5 : Construire une vraie marque (multi-plateforme)
- La boutique Dark Forest a aussi :
  - Un **site web Shopify** (cartes alu, plastique, lin + accessoires)
  - Des **pubs Google/YouTube** pour driver du trafic vers le site
  - Des **YouTubeurs** qui font des reviews/unboxing spontanement
- Preuve sociale sur le site (notifications d'achats en temps reel)
- Le site web seul pourrait generer autant voire plus qu'Etsy

### Etape 6 : Bonus - Vendre des lectures de tarot (100% digital)
- Niche decouverte : "tarot readings" sur Etsy
- Top vendeur : **$9 584/mois**, 320 ventes/mois, prix $30
- Boutique "Gypsy" : lecture tarot amour le jour meme
- Livraison = simple message Etsy (zero cout)
- **Marge : 87%** = $18,25 net par vente
- $58 176 en 10 mois = **$5 800/mois de profit pur**
- UNE seule annonce suffit

## Tips cles
- **Niche = marque** : se concentrer sur UN type de client et developper toute une gamme (cartes, boites, plateaux, accessoires)
- **Promo permanente -70%** : technique utilisee par les gros vendeurs Etsy pour attirer (le prix "barre" cree l'urgence)
- **Print on demand existe pour TOUT** : si le produit n'est pas chez Printify, chercher "[produit] print on demand" sur Google
- **Les YouTubeurs sont vos meilleurs commerciaux** : envoyer des produits gratuits a des createurs = reviews gratuites qui generent du trafic
- **La lecture de tarot digitale est un filon sous-exploite** : zero cout, marge 87%, livraison par message Etsy
- **Multi-plateforme obligatoire** : Etsy + site web propre + Google Ads = ne pas mettre tous ses oeufs dans le meme panier
- **Les donnees avant l'intuition** : utiliser Everbee pour valider la demande AVANT de creer un produit

## Pourquoi cette niche
- Parents, enseignants et homeschoolers cherchent des outils éducatifs créatifs
- La demande est forte, surtout chez les preschoolers et homeschoolers
- Les produits se vendent souvent en bundles → panier moyen élevé
- Digital downloads = zéro coût d'impression/livraison

## Types de produits qui marchent

| Type | Exemples |
|------|----------|
| Worksheets imprimables | Maths, lecture, sciences par tranche d'âge |
| Flashcards | Alphabet, chiffres, formes, sight words, bilingues |
| Produits interactifs | PowerPoints éditables, jeux PDF pour préscolaire |
| Kits physiques | Jouets Montessori, activity boxes |
| Posters | Maps, growth charts, ABC charts, classroom decor |
| Custom | Name tracing boards, lesson plans personnalisés |

## Process étape par étape

### Phase 1 — Recherche avec EverBee
1. Installer l'extension Chrome **EverBee** (gratuit)
2. Chercher sur Etsy : "alphabet flashcards", "kids worksheets", etc.
3. EverBee montre : recherches mensuelles, ventes estimées, revenus, keywords
4. Utiliser le **keyword tool** : chercher les keywords à haut volume + faible concurrence
5. Ex : "editable preschool worksheets" ou "Montessori toys for toddlers" = trending mais pas saturé

### Phase 2 — Créer les produits
1. Utiliser **Canva** pour créer les designs (worksheets, flashcards, posters)
2. Thèmes qui marchent : animaux, espace, holidays, lettres, chiffres
3. Créer des bundles (pack de 20 worksheets > 1 worksheet individuel)
4. S'assurer que c'est adapté à la tranche d'âge ciblée

### Phase 3 — Lister sur Etsy
1. Mockups professionnels montrant le produit en contexte
2. Titre SEO avec keywords validés par EverBee
3. 13 tags maximum, tous pertinents
4. Description détaillée : ce qui est inclus, tranche d'âge, usage
5. Sélectionner "Digital files" comme type de produit

## Process étape par étape

### Phase 1 — Créer le template link Canva
1. Ouvrir le design dans Canva
2. Cliquer **Share** (en haut à droite)
3. **NE PAS** cliquer "Copy link" (ça partage le design original !)
4. Cliquer **"See all"** → puis **"Template link"**
5. Cliquer **"Create template link"** → puis **Copy**
6. Vérifier que le statut dit "Live"

### Phase 2 — Créer le PDF cliquable pour Etsy
1. Faire une **copie** du design original (clic droit → Make a copy)
2. Modifier la copie : ajouter un bouton "Click here to customize"
3. Sélectionner le bouton → 3 dots → **Link** → coller le template link
4. Faire pareil avec le texte (double protection)
5. **CRUCIAL :** Exporter en **PDF Standard** (pas PNG !) → sinon le lien ne sera pas cliquable
6. Tester le PDF : ouvrir, cliquer le bouton, vérifier la redirection

### Phase 3 — Lister sur Etsy
1. Etsy Shop Manager → New listing
2. Add core details → sélectionner **"Digital files"**
3. Uploader le PDF comme fichier digital
4. Le client pourra le télécharger immédiatement après achat (100% passif)
5. Ajouter titre SEO, description, keywords, prix
6. Publier

## Outils mentionnés
- **Canva Pro** (ou gratuit) — création des templates
- **Etsy** — plateforme de vente

## Tips clés
- **JAMAIS** partager le lien direct du design → toujours un template link
- Le format **PDF** est obligatoire pour que les liens soient cliquables
- Optimiser le SEO Etsy (titre + tags) est crucial pour être trouvé
- Les listing covers de haute qualité augmentent le taux de conversion

## Source YouTube
- Titre : How I Made $500,000 Selling 1 Notion Template
- URL : https://www.youtube.com/watch?v=keOebOEv-Oc
- Chaine : Mike and Matty
- Vues : 30 472

## Process etape par etape

### Phase 1 : Trouver le bon probleme a resoudre
1. **Ne PAS commencer par "quel template creer"** : la bonne question est "quel probleme precis puis-je resoudre ?".
2. **Choisir un probleme que VOUS vivez** : ca le rend plus authentique, vous comprenez la douleur du client, et vous communiquez mieux.
3. **Exemple de l'auteur** : etudiant en medecine deborde entre etudes, creation YouTube, startup tech et vie perso. Il a cree "Extended Brain" par necessite, pas pour vendre.
4. **Cibler les debutants Notion** : c'est le coeur de marche. Les acheteurs de templates sont majoritairement des novices. Pas besoin de fonctionnalites avancees (formules, IA, etc.).

### Phase 2 : Construire le template
5. **Identifier les 3 elements cles** pour aller du point A au point B (pas plus).
6. **Creer un framework** : transformer ces 3 elements en un cadre conceptuel avec un nom et un visuel. Exemple : "Extended Brain" = Methodes (etudes) + Focus (taches) + Mindset (suivi de progression). Les frameworks positionnent comme expert.
7. **Eliminer TOUT le reste** : supprimer impitoyablement tout ce qui ne correspond pas au framework. L'auteur a supprime le suivi d'entrainement, le planificateur de repas, la gamification, etc.
8. **Simplifier au maximum** : titres clairs au-dessus de chaque section. Boutons d'action rapide. Instructions integrees dans le template. Creer une version "Easy" pour les debutants complets.
9. **Lecon cruciale** : la 1ere version de l'auteur etait trop complexe — premier client a demande un remboursement en 30 minutes. "La simplicite seduit, la confusion repousse."

### Phase 3 : Creer une experience delightful
10. **Parcours client complet** : page d'accueil > page de paiement > email de remerciement + invitation communaute > message personnalise a chaque acheteur > forum de discussion > service client dedie.
11. **Creer des tutoriels video** : montrer a l'ecran comment utiliser le template pour resoudre LE probleme (pas juste comment utiliser Notion). C'est l'avantage competitif n.1.
12. **Construire une communaute** : utiliser Circle (ou equivalent) pour creer un espace d'entraide, de cours et d'evenements.
13. **Demander les avis APRES tout ca** : ne demander un avis qu'apres avoir cree une vraie experience positive. Resultat : quasi 100% d'avis 5 etoiles.

### Phase 4 : Attirer l'attention et vendre
14. **Marketplaces** : Etsy, Notion Marketplace, Gumroad — visibilite integree mais limitee (le client doit deja connaitre le produit).
15. **YouTube (canal principal)** : creer 2 types de contenu :
    - **Cas d'usage** : montrer comment resoudre un probleme avec le template (ex: "comment organiser ses etudes")
    - **Contenu educatif** : enseigner Notion en utilisant votre template comme exemple
16. **Bouche-a-oreille** : devenir affilie Notion, parler lors de conferences (Make with Notion), parler de vos produits partout.
17. **Creer un lead magnet GRATUIT** : extraire un petit morceau du template payant et le distribuer gratuitement. L'auteur a cree un "Study Planner" gratuit = template d'etudes n.1 sur Notion, 800 000+ telechargements. Il genere la confiance avant la vente.
18. **Upsell naturel** : le lead magnet gratuit passe par la meme sequence de "delight" (email, communaute, tutoriels). Puis proposer la version payante avec plus de fonctionnalites.

## Outils mentionnes

- **YouTube** : canal principal d'acquisition de clients
- **Etsy / Notion Marketplace / Gumroad** : marketplaces de vente
- **Gumroad ou Payhip** : pages de paiement pour produits digitaux

## Tips cles
- **"Si vous ne pouvez pas l'expliquer simplement, vous ne le comprenez pas assez bien" (Einstein)** : la complexite est l'ennemi. Les debutants veulent de la simplicite.
- **Un seul produit, un seul canal** : ne pas se disperser. Devenir excellent pour aider un type de personne a resoudre un probleme.
- **Ne PAS copier les gros createurs** : vous vous ferez ecraser. C'est la precision du probleme resolu qui differencie.
- **Les 24 premieres heures avec un produit sont decisives** : si ca ne libere pas de dopamine, c'est un remboursement.
- **Traiter chaque acheteur comme un novice total** qui n'a jamais ouvert Notion.
- **Les tutoriels video sont le gold standard** : largement superieur aux instructions ecrites.
- **Le template n'est qu'un outil** : les gens achetent la resolution de leur probleme, pas un fichier Notion.
- **Le gratuit construit la confiance** : distribuer un lead magnet qui offre une vraie experience complete AVANT de vendre.
- **L'experience client differencie plus que le produit** : message personnalise, communaute, tutoriels, support — c'est ca qui genere les avis 5 etoiles et le bouche-a-oreille.

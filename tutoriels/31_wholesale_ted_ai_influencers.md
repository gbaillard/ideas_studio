## Idées business mentionnées

### Idée 1 : Micro-influenceur IA monétisé via TikTok Creator Rewards

- **Description** : Créer un personnage IA réaliste (pas un mannequin parfait, mais une personne "normale" et relatable) qui donne des conseils dans une niche spécifique (parentalité, finances, fitness, etc.). Produire des vidéos montage (assemblage de clips courts de 8 secondes) et les monétiser via le programme TikTok Creator Rewards (vidéos de 1 min+). Le format montage est idéal car les générateurs IA ne produisent que des clips courts.
- **Outils** :
  - **Leonardo AI** (gratuit, crédits quotidiens) : Génération de la photo du micro-influenceur avec le modèle Flux
  - **ChatGPT** (gratuit) : Écriture des scripts vidéo structurés en clips de 8 secondes max
  - **Google VO3 via Gemini Pro** (essai gratuit 1 mois) : Transformation de la photo + script en clips vidéo avec voix
  - **CapCut** (gratuit) : Montage vidéo, jump cuts, suppression des silences
  - **ElevenLabs** (optionnel, ~5$) : Uniformisation de la voix si VO3 change aléatoirement de voix
  - **CloudConvert** (gratuit) : Extraction audio MP3 depuis fichier vidéo si besoin
- **Revenus mentionnés** :
  - TikTok RPM moyen : ~1,20$/1 000 vues (exemple Reddit d'un créateur IA)
  - Exemple : 3,7 millions de vues = ~3 700$ pour une seule vidéo
  - Créateur Reddit avec influenceur IA "mère au foyer" + boutique Etsy : 2 000$/mois en produits digitaux (planners, templates Canva)
- **Process étape par étape** :
  1. **Créer la photo de l'influenceur** :
     - Ouvrir Leonardo AI, sélectionner le modèle Flux, format mobile
     - Prompt : inclure "selfie mode framing", ajouter des imperfections (taches de rousseur, "looks tired") pour le réalisme
     - Astuce : demander à ChatGPT de créer le prompt en uploadant une photo de référence d'un vrai influenceur TikTok
     - Upscaler l'image et télécharger
  2. **Écrire le script** :
     - Sur ChatGPT, demander un script structuré en clips de max 8 secondes
     - Uploader un exemple de script existant comme référence de style
     - Pour TikTok : script de 1 min minimum. Pour YouTube/Etsy : pas de durée minimale
  3. **Générer les clips vidéo** :
     - Dans Gemini Pro (essai gratuit), activer "Videos with VO3"
     - Uploader la photo de l'influenceur
     - Prompt : demander du photoréalisme, éclairage naturel, mouvements d'yeux naturels, préciser le ton de la voix
     - Taper le texte du script pour chaque clip
     - Chaque clip prend 2-3 min à générer
     - Avec l'essai gratuit : 3 clips/jour (~24 sec de contenu)
     - Astuce : combiner plusieurs phrases par clip pour maximiser le contenu
  4. **Monter la vidéo** :
     - Importer tous les clips dans CapCut
     - Faire des jump cuts : supprimer les silences et les parties non naturelles
     - Technique : placer le curseur, split, sélectionner le passage indésirable, supprimer
     - Réorganiser les clips dans l'ordre souhaité
     - Exporter en MP4 haute qualité
  5. **Optionnel - Uniformiser la voix** (si VO3 change de voix entre les clips) :
     - Exporter la vidéo depuis CapCut
     - Extraire l'audio MP3 via CloudConvert
     - Importer dans ElevenLabs Voice Changer
     - Créer une voix custom : décrire les imperfections souhaitées (ex: "recorded on a phone microphone")
     - Générer 3 options, choisir, nommer et sauvegarder
     - Télécharger le nouveau fichier audio
     - Dans CapCut : muter tous les clips, ajouter le nouvel audio
     - Exporter la version finale
  - **Cadence de production** :
    - Gemini Pro gratuit (3 clips/jour) : 1 vidéo courte/jour (YouTube/Etsy) OU 1 vidéo TikTok 1min+ tous les 2-3 jours
    - Plan Ultra : production plus intensive possible

- **Revenus mentionnés** : Non précisés spécifiquement

### Idée 3 : Monétisation via Etsy (produits digitaux)

- **Description** : Construire une audience avec l'influenceur IA, puis vendre des produits digitaux (planners, templates Canva) sur Etsy
- **Outils** : Etsy, Canva
- **Revenus mentionnés** : 2 000$/mois (exemple Reddit)

## Tips clés de la créatrice

- La clé du réalisme est de créer un influenceur qui a l'air "normal" et imparfait, PAS parfait -- ajouter des taches de rousseur, un air fatigué dans le prompt
- Le format "montage" (assemblage de clips courts) est parfait car c'est déjà un format populaire sur TikTok (exemple : Davey qui filme toujours dans la même pose/tenue)
- On peut réutiliser la même photo pour plein de vidéos différentes (donne l'illusion de batch filming)
- Pour garder une voix consistante avec VO3, créer tous les clips dans le même chat thread (même seed)
- Les voix par défaut de ElevenLabs sont trop "professionnelles" pour un micro-influenceur amateur -- créer une voix custom avec des imperfections
- Le business a évolué : on ne parle plus d'OnlyFans avec des modèles IA, mais de vrais contenus utiles avec des personnages relatables
- Les jump cuts sont essentiels pour le réalisme et le dynamisme du montage final

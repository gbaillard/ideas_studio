## Process étape par étape

### Recherche de marché
1. Chercher des storybooks sur Amazon qui se vendent bien (ex : "Train Your Angry Dragon" BSR 3 773)
2. Les story books peuvent se vendre en 4 formats : paperback, hardcover, Kindle, audiobook = 4 sources de revenus
3. Vérifier la constance des ventes (graphe "all time" pour confirmer que ça vend toute l'année)
4. Analyser les intérieurs via les samples et reviews : noter la structure (illustration + texte, chapitres, etc.)

### Création avec Ohanashi AI Gen
5. Aller sur Ohanashi AI Gen (ohanashi.ai)
6. Configurer les paramètres :
   - **Tranche d'âge** : 5-10 ans (ou preschool, adult, etc.)
   - **Langue** : anglais (aussi dispo en espagnol, français, allemand, italien)
   - **Plot outline** : écrire une idée ou cliquer "Give me an idea" pour que l'IA propose un scénario complet
   - **Nom d'auteur** : son nom ou un pen name
   - **Style artistique** : "Children's book illustration" (aussi dispo : watercolor, vintage, modern minimalist, etc.)
   - **Style d'écriture** : descriptive, action-packed, conversational, etc.
   - **Personnages** : laisser l'IA générer noms + descriptions + images, ou personnaliser
7. Paramètres avancés :
   - **Word count** : automatique (~130 mots/page) ou manuel
   - **Fréquence des images** : image après chaque page (recommandé)
   - **Cocher "Use Nano Banana"** = meilleur générateur d'images
   - **Variations d'images** : 2 par page pour pouvoir choisir
   - **Auto-générer la couverture** avec titre et nom d'auteur
   - **Format** : 8.5 x 8.5 inches (uniquement print) ou 6 x 9 inches (compatible Kindle + paperback)
8. Cliquer "Generate Story" — attendre ~10 minutes

### Review et ajustements
9. Vérifier la story preview : couverture, illustrations, texte de chaque page
10. Pour chaque page, choisir entre variation 1 et 2 des illustrations
11. Sauvegarder les sélections ("Save Page")
12. Relire l'histoire complète pour vérifier cohérence, erreurs, contenu inapproprié
13. Ajuster le nombre de pages si nécessaire

### Export pour KDP
14. Cliquer "Export Story"
15. Configurer :
    - Trim size (8.5 x 8.5 ou 6 x 9)
    - Marges (top, bottom, left, right)
    - Title page avec personnage choisi
    - Copyright page (personnaliser)
    - Police (taille 18 par défaut, plusieurs options)
    - Placement images : before page, top, left, right, after page
    - Format : PDF pour KDP
    - Options : include page numbers, add bleed
16. Exporter et télécharger le PDF
17. Uploader sur KDP (interior file + cover file)

## Outils mentionnés
- **Ohanashi AI Gen** (ohanashi.ai) — outil IA tout-en-un pour storybooks (histoire + illustrations + couverture + export PDF)
- **Nano Banana** — modèle de génération d'images intégré dans Ohanashi
- **Titans Pro / Titans Quick View** — extension Chrome pour analyse de niche Amazon
- **Amazon KDP** — plateforme de publication
- **Etsy** — plateforme de vente alternative

## Chiffres clés
- "Train Your Angry Dragon" : BSR 3 773, royalties $5 000-$8 000/mois, 1 000-2 000 ventes/mois
- Livre de puns/humour : BSR 59 623, royalties $600-$1 000/mois, 120-180 ventes/mois
- "Axolottal Tails" (indépendant) : BSR 16 350, royalties $1 000-$1 600/mois, 400-612 ventes/mois, prix $9.99
- Temps de création : ~10 minutes pour un storybook complet
- 3 plans tarifaires : basic, middle, elite (full unlock avec audiobook + cover generator)

## Tips clés
- **Format 6 x 9 > 8.5 x 8.5** si on veut vendre en Kindle + paperback (plus de canaux de vente)
- Les storybooks se vendent toute l'année (pas saisonnier)
- 4 formats de vente possibles = 4x les revenus potentiels sur un seul livre
- Toujours relire et vérifier le contenu généré par l'IA avant publication
- Si on utilise d'autres langues (espagnol, français...), faire vérifier la traduction par un natif
- Utiliser "Give me an idea" si on n'a pas d'inspiration — l'IA propose des scénarios complets
- Ohanashi peut aussi générer la version audiobook du storybook (plan elite)
- Structure type : title page > copyright > illustration > texte > illustration > texte...

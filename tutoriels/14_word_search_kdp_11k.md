## Process étape par étape

### Phase 1 — Comprendre le timing Q4
- Les livres d'activités de Noël **explosent** en octobre-décembre
- Le créateur a commencé en octobre 2024 : $100 → $900 → $3 000 en 3 mois
- Tous ses livres étaient publiés avec IA

### Phase 2 — Trouver la niche saisonnière
1. Chercher sur Amazon : "Christmas word search for adults/kids/seniors"
2. Repérer les livres auto-publiés avec de bonnes ventes
3. Exemples trouvés :
   - "Cozy Christmas Word Search for Adults" → $172/jour en royalties
   - Un autre livre Christmas word search → $10 000+/mois en revenu
4. **Stratégie clé :** reprendre le mot-clé principal + ajouter un adjectif
   - "Relaxing Christmas Word Search for Seniors"
   - "Comfy Christmas Word Search for Kids"
   - "Festive Holiday Word Search Large Print"

### Phase 3 — Créer le livre
1. Utiliser **LitPal.ai** → Word Search generator (ou Book Bolt)
2. Paramètres : thème Christmas, 80-100 pages, format 8.5×11"
3. Personnaliser les listes de mots par thème saisonnier
4. Générer l'intérieur automatiquement

### Phase 4 — Cover + Publication
1. Créer la cover avec thème Noël (rouge/vert dominant)
2. Titre = mot-clé exact ciblé
3. Publisher sur KDP → prix $9.99
4. **Publier AVANT fin octobre** pour profiter du pic décembre

## Chiffres documentés
- Livre Christmas word search top : **$11 000+/mois** en revenu
- Le créateur perso : **$3 000 en décembre** (son 3ème mois)
- Royalty à $9.99 : ~$3.15 par vente
- BSR 4700 = ventes très solides

## Niches saisonnières à exploiter
- Christmas word search (adultes, seniors, kids)
- Halloween activity books (publier en septembre)
- Valentine's Day word search (publier en janvier)
- Easter activity books (publier en février)
- Back to School (publier en juillet)

## Tips clés
- **Le timing est TOUT** : publier 6-8 semaines avant la fête
- Le titre doit contenir le **mot-clé exact** que les acheteurs cherchent
- Les covers rouge/vert pour Noël convertissent mieux
- Combiner : word search + coloring + mazes dans un activity book = valeur perçue maximale
- Q4 peut représenter 60-70% du revenu annuel d'un éditeur KDP

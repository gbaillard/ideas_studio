## Process étape par étape

### Recherche de niche
1. Chercher sur Amazon un type de coloring book (ex : "cute cat coloring book bold and easy")
2. Vérifier le nombre de résultats : viser 1 000 ou moins = compétition faible
3. Identifier 3+ livres KDP avec un BSR (bestsellers rank) de 300 000 ou moins
4. Vérifier le niche score (viser 50+) et le Titans Pro Analysis (demande + opportunité)
5. Regarder les intérieurs des livres concurrents pour s'inspirer (via samples, A+ content, reviews)
6. Calculer les royalties potentielles via KDP Printing Cost & Royalty Calculator

### Création du livre avec KD
7. Créer un nouveau projet dans KD aux dimensions du livre (ex : 8.5 x 8.5 inches), mettre DPI à 300
8. Cliquer sur l'artboard > "Smartboard" pour créer un Smartboard (outil beta)
9. **Méthode Smartboard :**
   - Commencer avec un artboard vierge
   - Créer des Smartboards depuis ce blank avec des prompts descriptifs
   - Prompt type : "black and white coloring page, bold and easy, of a cute cat playing in the kitchen"
   - Chaque Smartboard génère une page unique en ~1 minute
10. S'inspirer des intérieurs des concurrents pour écrire les prompts (ex : "cat wrapped in Christmas lights with Christmas tree in background")
11. Créer des variations depuis un Smartboard existant (ex : changer le thème Halloween, ajouter des éléments)
12. Si le résultat ne convient pas, régénérer

### Couverture
13. Utiliser un Smartboard avec un prompt "full color" pour générer la couverture
14. Ne PAS mettre "book cover" dans le prompt (l'IA crée littéralement un livre)
15. Ajouter le titre soit via le prompt, soit via les outils texte de KD (plus de flexibilité)

### Pages structurelles
16. Ajouter une page "This book belongs to" avec les outils texte de KD
17. Ajouter des pages blanches entre chaque illustration (recto verso)
18. Organiser : couverture > book belongs to > page blanche > illustration > page blanche > illustration...

### Export
19. Download > All Artboards > PDF > Combine all artboards into one file > Optimize quality
20. Uploader sur KDP

## Outils mentionnés
- **KD** (kd.ai) — plateforme de design + IA avec Smartboard
- **Titans Pro / Titans Quick View** — extension Chrome pour analyse de niche Amazon KDP
- **KDP Printing Cost & Royalty Calculator** — calculateur officiel Amazon

## Chiffres clés
- Exemple : "Cozy Cats" coloring book, BSR 21 468, royalties $7 à $1 089/mois, 336-504 ventes/mois
- Prix de vente : $9.99 (marge confortable pour ce type de livre)
- Estimation calculée : 504 ventes x $3.15 royalty = $1 587/mois
- Tokens KD : 42 tokens par image haute qualité, pack 2 000 tokens à $12
- Code promo : "JeffYT" pour 25% sur KD (nouveaux utilisateurs)

## Tips clés
- **Bold and Easy** = style très populaire et vendeur (peu de détails fins, gros traits)
- Objectif : ~40 illustrations par livre
- Utiliser les intérieurs des concurrents comme inspiration pour les prompts (pas copier, s'inspirer du thème)
- Chaque génération prend ~1 minute, donc un livre complet en moins d'1 heure
- Les Smartboards permettent de créer des variations infinies à partir d'une seule image de base
- Toujours vérifier les images pour des erreurs avant d'uploader

## Process étape par étape

### Phase 1 — Trouver un niche croisé
1. Prendre une niche large profitable (ex : Teaching, Nursing)
2. **Cross-niching** : croiser avec une autre niche (Western, Reading, Pride, Hiking, Holidays, Moms/Dads)
3. Ex : "Teaching" + "Western" = "Teaching Ain't My First Rodeo" → peu de concurrence
4. Utiliser **ChatGPT** pour générer des phrases de t-shirts pour la niche croisée

### Phase 2 — Créer les designs avec KD (Kittl)
1. Ouvrir **KD** (plateforme design POD spécialisée)
2. Chercher un **template** correspondant au thème (ex : "western", "groovy", "pride")
3. **Modifier le texte** : remplacer par la phrase de ta niche
4. Le texte s'auto-formate (courbes, waves, styles du template)
5. Ajouter des **illustrations** depuis la bibliothèque si besoin
6. Ajuster les **couleurs** avec les palettes
7. Vérifier : texture "masked", taille **4500px minimum**
8. Download avec **background removed**

### Phase 3 — Uploader sur Printify
1. **Printify** → Catalog → T-shirts → Bella+Canvas 3001
2. Upload le design → ajuster la position
3. Sélectionner 2-3 variantes de couleurs (pas trop)
4. Sauvegarder le produit

### Phase 4 — Lister sur Etsy
1. Printify synchro automatiquement vers Etsy
2. **Titre SEO** : penser aux recherches clients ("best St. Patrick's Day gift for teachers")
3. **Tags** : 13 maximum, retirer les trop génériques, garder les spécifiques
4. **Prix** : $21.99 (standard), $24.99 (2XL/3XL)
5. Ajouter un **mockup avec personne** (via Placeit) comme image principale
6. Désactiver Etsy Ads pour les nouvelles listings
7. Mettre en renouvellement **manuel** (pour optimiser avant re-publish)

### Exemples de niches croisées réalisées
1. Teaching + Western = "Teaching Ain't My First Rodeo"
2. Teaching + Reading = "Books, Coffee, Teach, Repeat"
3. Teaching + Pride = "Proud to Teach"
4. Teaching + Retro = "One Groovy Teacher"
5. Teaching + Dad = "Teaching Pays the Bills, Dad Jokes Free"
6. Teaching + Mom = "Teacher by Day, Mom by Night"
7. Teaching + Hiking = "My Students Are Calling and I Must Go"
8. Teaching + St. Patrick's = "Lucky to Teach"

## Outils mentionnés
- **KD (Kittl)** — plateforme design POD avec templates
- **Printify** — print on demand (gratuit)
- **Etsy** — marketplace
- **Placeit** — mockups avec personnes
- **ChatGPT** — brainstorming phrases

## Tips clés
- **Templates** = 4-10x plus rapide que from scratch
- Chaque design devrait prendre **~1 minute** max avec un template
- Si un design se vend bien → faire **10-20 variations** du même phrase
- Taille minimum : **4500px** pour qualité d'impression
- Les designs textuels simples marchent souvent mieux que les designs complexes
- Le **cross-niching** est la clé pour éviter la saturation

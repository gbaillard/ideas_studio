# KDP Coloring Books avec Claude + LitPal IA

## Source YouTube
- **Titre :** How to Publish PROFITABLE KDP Coloring Books Using Claude + AI (Step-by-Step)
- **URL :** https://www.youtube.com/watch?v=R1VVAGTcJWM
- **Chaîne :** Zachary Donovan | **Vues :** 10 095

## Process étape par étape

### Étape 1 — Trouver une niche profitable avec Claude
1. Ouvrir **Claude** (modèle Opus 4.6)
2. Coller un prompt de recherche KDP niche (disponible dans sa communauté School)
3. Claude génère **20 niches** avec analyse de la concurrence Amazon
4. Claude produit aussi : titres optimisés, keywords KDP, descriptions produit, pricing

### Étape 2 — Valider la niche sur Amazon
1. Copier la niche trouvée par Claude (ex : "90s nostalgia coloring book")
2. Chercher sur Amazon → vérifier qu'il y a des livres auto-publiés qui vendent
3. Vérifier les revenus estimés (ex : $600/mois pour un livre dans cette niche)

### Étape 3 — Générer le livre de coloriage
1. Aller sur **LitPal.ai** → Dashboard → Coloring Books → Create New
2. Remplir : titre, description (générer avec IA), style (Bold and Easy), nb pages (40)
3. Cliquer **Generate Coloring Book** → attendre quelques minutes
4. Résultat : 40 pages de coloriage illustrées, détaillées, prêtes à imprimer

### Étape 4 — Créer la couverture
1. LitPal.ai → **Book Covers Generator** → Create New Cover
2. Paramètres : Coloring Book, 8.5×8.5, titre (utiliser celui de Claude), auteur, 80 pages, Paperback
3. Style : Bold and Easy (cohérent avec l'intérieur)
4. Générer → exporter en PDF KDP-ready

### Étape 5 — Publier sur Amazon KDP
1. Amazon KDP → Create New Title → Paperback
2. Copier-coller depuis Claude : titre, sous-titre, description, keywords
3. Uploader le manuscrit PDF + la couverture PDF
4. **Déclarer l'utilisation de l'IA** (obligatoire)
5. Lancer le previewer → vérifier le rendu
6. Prix recommandé : **$9.99** → royalty $3.15/vente
7. Publier

## Outils mentionnés
- **Claude Opus 4.6** — recherche niche + keywords + descriptions ($20/mois)
- **LitPal.ai** — génération coloring book + couverture (crédits, ~85 crédits pour 40 pages)
- **Amazon KDP** — publication (gratuit)

## Chiffres clés
- Niche "90s nostalgia" : livres concurrents font $600/mois
- Prix optimal : $9.99 → $3.15 royalty/vente
- Temps total : ~15-20 minutes du début à la publication

## Tips clés
- Utiliser Claude pour la **recherche de niche** = bien plus rapide et complet que la recherche manuelle
- Claude fournit aussi les **Amazon Ad keywords** pour PPC
- LitPal gère l'intérieur ET la couverture en un seul outil
- Toujours **disclosure IA** à l'upload KDP

# Chatbots IA pour Entreprises Locales

## Source YouTube
- **Titre :** Create AI Chatbots for Local Businesses (Even If You're a Beginner)
- **URL :** https://www.youtube.com/watch?v=OJPK81VmjiE
- **Chaîne :** Jason Wardrop | **Vues :** 58 730

## Process étape par étape

### Phase 1 — Setup du chat widget sur le site client
1. Utiliser un logiciel d'automatisation IA (type GoHighLevel)
2. Aller dans **Chat Widget** → Create New → sélectionner "Live Chat"
3. Personnaliser couleurs et style
4. Cliquer **Get Code** → copier le code
5. Coller dans le **body/footer** du site web du client
6. Résultat : un widget "Hi! Have a question? Text us here" apparaît en bas à droite

### Phase 2 — Configurer le bot IA (texte)
1. Settings → **Conversation AI** → mettre sur **Autopilot**
2. Activer les canaux : Instagram, Facebook, SMS, Chat widget, Live chat
3. Paramétrer : délai de réponse (1 min), nombre max de messages

### Phase 3 — Entraîner le bot
1. **Bot Training** → coller l'URL du site web client
2. Cliquer **Get Data** → le bot scanne tout le site en 10-20 secondes
3. Il apprend : services, horaires, prix, FAQ
4. Ajouter les **FAQ manuellement** si des questions récurrentes ne sont pas sur le site
5. **Bot Trial** : tester les réponses, thumbs up/down pour corriger
6. Si réponse incorrecte → renseigner la bonne réponse → "Train Bot"

### Phase 4 — Configurer l'objectif du bot
1. **Configure Intent** : choisir entre rediriger vers une page web OU **booker un RDV**
2. Sélectionner le calendrier de booking
3. Le bot guide la conversation vers la prise de RDV automatiquement

### Phase 5 — Ajouter l'IA Voice (appels téléphoniques)
1. Settings → **AI Agent** → Create Agent
2. Donner un nom à l'agent (ex : "Sally")
3. Choisir une voix parmi les options disponibles
4. Personnaliser le message d'accueil
5. Définir les objectifs : capturer nom, email, raison de l'appel
6. Associer un numéro de téléphone
7. L'IA répond à 100% des appels entrants (62% des appels vont normalement sans réponse chez les PME)

### Phase 6 — Vendre le service
- Setup unique : $1 500-3 000
- Retainer mensuel : $200-500/mois
- Cible : plombiers, chiropracteurs, avocats, dentistes, massage therapists
- Argument : "62% des appels entrants sont perdus. Mon bot répond 24/7."

## Outils mentionnés
- **GoHighLevel** (ou similaire) — plateforme tout-en-un (CRM + chatbot + AI voice + website)
- Alternatives : **Botpress** (gratuit), **Voiceflow**, **Chatbase**

## Chiffres clés
- Les sites web convertissent < 1% des visiteurs → le chat widget booste ce taux
- 62% des appels entrants aux PME restent sans réponse
- Le bot se configure en **quelques minutes**, pas besoin de coder

## Tips clés
- Commencer par le **chat widget** (le plus simple et visible)
- L'IA voice (téléphone) est le **gros différenciateur** → peu de concurrents offrent ça
- Toujours **tester le bot** avant de le livrer au client
- Les PME locales sont le marché idéal : elles ont le problème ET le budget
- Le business model est **setup + retainer** = revenu récurrent

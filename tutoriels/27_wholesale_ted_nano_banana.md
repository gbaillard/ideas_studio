## Idees business mentionnees

### Idee 1 : T-shirts a blagues homophoniques
- **Description** : Creer des t-shirts avec des jeux de mots/blagues homophoniques (mots qui sonnent pareil mais sens different, souvent a double sens). Style tres viral en ce moment.
- **Outils** :
  - ChatGPT : brainstormer des blagues homophoniques dans differentes niches
  - Nano Banana (Google AI Studio) : generer les images de personnages/animaux avec image de reference
  - Canva Pro : combiner image + texte, upscaler, supprimer fond
- **Revenus mentionnes** :
  - Un t-shirt similaire sur Amazon : 3,000+ ventes/mois = $59,000 a $79,000/mois
  - Un t-shirt sur Etsy : 7 ventes/jour = $180+/jour
- **Process etape par etape** :
  1. Reperer un design de blague homophonique viral sur Etsy/Amazon
  2. Utiliser ChatGPT pour brainstormer de nouvelles blagues dans d'autres niches
  3. Ouvrir Nano Banana dans Google AI Studio
  4. Uploader une image de reference (style, expression faciale)
  5. Demander a Nano Banana de creer un nouvel animal/personnage dans le meme style
  6. Demander des modifications (position assise, autre pose) - Nano Banana garde le meme personnage
  7. Combiner image + texte dans Canva avec une police vintage/distressed (ex: Edmund Texture)
  8. Upscaler l'image avec l'outil Canva AI
  9. Supprimer le fond avec Background Remover
  10. Exporter en PNG transparent haute resolution

### Idee 2 : T-shirts "dramatiques" viraux (style 3 loups/lune)
- **Description** : Recreer le style de t-shirt viral avec des animaux/personnages dans des poses dramatiques avec lune et etoiles (inspire du fameux "three wolf moon t-shirt"). Les gens les achetent ironiquement.
- **Outils** :
  - Nano Banana : generation avec image de reference pour le layout
  - Canva Pro : edition finale
- **Revenus mentionnes** :
  - Version Possums sur Amazon : 200+ ventes/mois
  - Version Grandmas : 50+ ventes/mois
  - Un store Etsy avec ce style : $580,000 en moins de 2 ans (estime par Alura)
  - Un t-shirt Etsy : $41,000 sur l'annee (estime par Alura)
- **Process etape par etape** :
  1. Uploader une image de reference du style dans Nano Banana
  2. Demander le meme layout mais avec un animal/theme different
  3. Iterer : si le resultat n'est pas parfait, relancer le meme prompt jusqu'a obtenir le bon resultat
  4. Editer et finaliser dans Canva

### Idee 3 : T-shirts art vintage modernise (Van Gogh, estampes japonaises)
- **Description** : Prendre des oeuvres d'art du domaine public (Van Gogh, La Grande Vague de Kanagawa) et les moderniser avec des elements fun (chat, panda, texte).
- **Outils** :
  - Wikipedia : telecharger des oeuvres d'art du domaine public gratuitement
  - Nano Banana : modifier l'oeuvre d'art, ajouter des elements, changer le texte
  - Canva Pro : distresser les bords, upscaler, fond transparent
- **Revenus mentionnes** :
  - T-shirt Van Gogh + chat noir sur Amazon : 700+ ventes/mois
  - Populaire aussi sur Etsy avec les styles Van Gogh et estampes japonaises
- **Process etape par etape** :
  1. Trouver une oeuvre d'art vintage sur Wikipedia (domaine public = usage commercial gratuit)
  2. Telecharger l'image
  3. Ouvrir Nano Banana, uploader l'oeuvre + une image de reference d'un t-shirt modernise
  4. Demander d'ajouter un element fun (chat surfant, panda, etc.)
  5. Demander des modifications (texte en japonais, changements de details)
  6. Creer un nouveau chat dans Nano Banana pour l'etape de distressing
  7. Uploader l'image + reference d'un t-shirt avec bords distresses
  8. Demander de distresser les bords
  9. Finaliser dans Canva : upscaler + fond transparent

## Tips cles de la creatrice
- Nano Banana est un "Photoshop IA" : il suit les prompts et references avec precision
- Avantage unique de Nano Banana : il peut modifier une image existante tout en gardant exactement le meme personnage/style (les autres outils IA ont du mal avec ca)
- Nano Banana ne reussit pas a chaque fois : quand ca rate, relancer le meme prompt jusqu'a obtenir le bon resultat
- Google AI Studio est accessible avec un compte Google gratuit
- Gemini Pro (version payante) disponible en essai gratuit 30 jours
- Toujours upscaler les images Nano Banana dans Canva avant impression
- Les oeuvres d'art anciennes (domaine public) sont une mine d'or gratuite pour les designs
- Utiliser Alura pour estimer les ventes et valider les niches sur Etsy
- Le style "ironique/dramatique" (3 wolves moon) est un filon : les gens achetent ces t-shirts pour rire

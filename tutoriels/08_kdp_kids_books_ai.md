## Process étape par étape

### Phase 1 — Trouver une niche profitable
**Méthode 1 : Résoudre un problème parent**
- Ex : enfant qui se ronge les ongles → livre sur pourquoi c'est mauvais
- Les parents achètent des livres qui **résolvent un problème comportemental**

**Méthode 2 : Sujets que les enfants trouvent intéressants**
- Océan, dinosaures, avions, espace, animaux
- Lier au thème de Noël pour le Q4 = explosion des ventes

### Phase 2 — Générer le livre avec LitPal.ai
1. **LitPal.ai** → Kids Book Generator
2. Remplir : titre, description, nom personnage principal, genre
3. Nombre de pages : 10-30 (standard industrie : **24 pages**)
4. Style illustration : Storybook, 3D Animation, Vintage, Pop Art, Realistic, Digital Painting
5. Option : uploader une **photo de référence** pour le personnage (ou laisser l'IA générer)
6. Générer → quelques minutes → livre complet (texte + illustrations cohérentes)

### Phase 3 — Mettre en page dans Canva
1. Utiliser le **template Canva gratuit** fourni par le créateur
2. Pour chaque page : télécharger l'image de LitPal + copier le texte
3. Placer image + texte dans le template
4. Répéter pour toutes les pages
5. Exporter en PDF

### Phase 4 — Créer la couverture
1. Utiliser le **template cover Canva** fourni
2. Réutiliser l'art de l'intérieur pour la couverture
3. Ajouter titre, nom auteur
4. Exporter en PDF aux dimensions KDP

### Phase 5 — Publier sur KDP
1. Amazon KDP → Create New Title → Paperback
2. Remplir titre, sous-titre
3. Uploader manuscrit PDF + couverture PDF
4. **Disclosure IA obligatoire**
5. Publier

## Outils mentionnés
- **LitPal.ai** — générateur kids books IA (~55-85 crédits selon pages, 25% off pour nouveaux)
- **Canva** — mise en page (templates gratuits fournis)
- **Amazon KDP** — publication (gratuit)

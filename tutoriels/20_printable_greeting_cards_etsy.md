## Process etape par etape
1. **Choisir un type de carte** : anniversaire, Saint-Valentin, fetes, etc. Identifier la niche qui vous plait.
2. **Telecharger des graphismes et polices** : utiliser Creative Fabrica (licence commerciale incluse dans l'abonnement) pour obtenir des illustrations et des fonts de qualite.
3. **Creer le recto dans Canva** : nouveau design > taille personnalisee 5x7 pouces (format standard carte de voeux). Importer les graphismes, ajouter du texte avec les polices telechargees. Jouer avec les couleurs, la transparence, le positionnement.
4. **Creer le verso** : dupliquer la page, ajouter des elements decoratifs (motifs, coeurs, etc.), rester dans la meme palette visuelle.
5. **Creer le fichier imprimable pour le client** : nouveau design au format Lettre US (8.5 x 11 pouces). Ajouter un rectangle de 7x10 pouces avec bordure pointillee (ligne de decoupe). Copier-coller recto et verso a l'interieur, bien centres. Le client pourra imprimer, decouper et plier.
6. **Exporter en PDF** : Partager > PDF Print > Aplatir le PDF > selectionner CMYK (meilleur pour l'impression). C'est le fichier que le client telechargera.
7. **Creer les images produit (mockups)** : nouveau design 2500x2000 pixels. Ajouter un fond texture (bois gris, etc.). Placer le JPEG de la carte avec une ombre portee. Ajouter une enveloppe. Ajouter du texte "Printable" et "Digital Download" de maniere tres visible.
8. **Creer une 2e image produit** : montrer recto + verso, indiquer "Print on US Letter size", "Cut & fold to 5x7 inch card".
9. **Ajouter des images d'information** : expliquer clairement au client ce qu'il recoit, comment decouper, comment plier (s'inspirer des meilleures boutiques).
10. **Publier sur Etsy** : Shop Manager > Listings > Add a listing. Selectionner "Digital" pour activer le telechargement numerique. Uploader le fichier PDF. Ajouter les images produit.
11. **Bonus : ajouter un fichier supplementaire** : carte de remerciement, code promo, ou instructions d'impression/decoupe. Les meilleurs vendeurs font ca.
12. **Alternative pour les graphismes** : utiliser Spark de Creative Fabrica (generateur IA). Ecrire un prompt descriptif ("a cute puppy wearing a birthday hat"), generer des images, les telecharger et les utiliser dans Canva.
13. **Proposer 2 tailles** : 5x7 pouces ET 4x6 pouces pour maximiser les ventes.

## Outils mentionnes
- **Canva** (version gratuite ou Pro avec essai 30 jours) : creation des designs, export PDF/JPEG
- **Creative Fabrica** : polices et graphismes avec licence commerciale (10 telechargerments gratuits avec le trial)
- **Creative Fabrica Spark** : generateur d'images IA par prompt (inclus dans le trial)
- **Etsy** : plateforme de vente
- **Adobe Illustrator** : mentionne comme alternative pour creer ses propres graphismes

## Tips cles
- **Toujours indiquer clairement "Digital Download"** sur les images produit : les clients doivent comprendre qu'il n'y a pas d'envoi physique
- **Aplatir le PDF** avant export : garantit une impression correcte
- **Exporter en CMYK** : meilleur rendu a l'impression (fonctionnalite Canva Pro)
- **Minimum 2000 pixels de large** pour les images Etsy (cote le plus court)
- **Ajouter un fichier bonus** (remerciement, code promo, instructions) : differenciation par rapport aux autres vendeurs
- **Proposer les 2 formats** (5x7 et 4x6) car les meilleures boutiques le font
- **Les prompts IA doivent etre tres specifiques** : plus c'est precis, meilleur est le resultat avec Spark
- **Lire les conditions de licence** de chaque element utilise dans Canva

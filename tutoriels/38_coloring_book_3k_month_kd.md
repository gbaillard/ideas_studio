# Créer un coloring book à $3 000/mois avec KD (personnages cohérents)
## Source YouTube
- Titre : I Created a $3,000/Month Coloring Book Using ONE Tool! (Full Tutorial)
- URL : https://www.youtube.com/watch?v=oW3WAwPaMog
- Chaîne : Moe Income
- Vues : 8 724

## Process étape par étape
1. Rechercher une niche sur Amazon/Etsy (ex : "cozy coloring book") — vérifier que les livres existants font de bonnes ventes
2. Ouvrir KD (plateforme de design avec IA) et créer un nouveau fichier 8.5 x 8.5 inches
3. Utiliser le chat IA intégré de KD pour générer des prompts de pages de coloriage (ex : coloring book pour filles)
4. Copier le prompt de la 1ère page et le coller dans le générateur d'images de KD
5. Choisir les dimensions 1:1, sélectionner le modèle "Nano Banana Pro" et générer
6. **Astuce clé — Smartboard pour la cohérence des personnages :**
   - Cliquer sur Artboard 1 > point violet à droite > créer un Smartboard lié
   - Le Smartboard modifie UNIQUEMENT ce qu'on lui demande sans altérer le reste de l'image
   - Chaque nouvelle page créée via Smartboard garde les mêmes personnages
7. Ajouter plusieurs Smartboards à l'Artboard 1, un par page, avec un prompt différent pour chaque scène
8. Générer toutes les pages en parallèle — le livre complet prend ~15 minutes
9. Télécharger les pages intérieures en PDF à 300 DPI

### Création de la couverture
10. Aller sur KDP Cover Calculator pour obtenir les dimensions exactes (selon format + nombre de pages)
11. Créer un nouveau design dans KD aux dimensions calculées
12. Télécharger le template depuis KDP Cover Calculator et l'utiliser comme guide
13. Choisir une page intérieure, créer un Smartboard et demander à KD de la coloriser avec des couleurs douces pour enfants
14. Placer l'image colorisée sur la couverture avant, ajouter des pages intérieures sur la couverture arrière
15. Laisser la zone jaune du barcode vide (obligatoire Amazon)
16. Ajouter un titre avec les polices pré-faites de KD, ajuster couleurs et tailles
17. Ajouter le nom d'auteur sur la couverture arrière
18. Exporter la couverture en PDF

## Outils mentionnés
- **KD** (kd.ai) — plateforme de design tout-en-un avec IA, chat IA, générateur d'images, Smartboard
- **Modèle Nano Banana Pro** — modèle de génération d'images dans KD
- **KDP Cover Calculator** — outil web pour calculer les dimensions exactes de couverture Amazon

## Chiffres clés
- Niche "cozy coloring book" : des milliers de ventes par jour sur Amazon
- Objectif : avoir au moins 5 livres dans une même niche
- Temps de création : ~15 minutes pour un livre complet avec couverture
- Code promo : "moincome YT" pour 25% de réduction sur KD

## Tips clés
- **Smartboard = game changer** : permet de garder les mêmes personnages sur toutes les pages (problème #1 des livres IA)
- Toujours exporter à 300 DPI pour la qualité d'impression
- Utiliser les pages intérieures comme éléments visuels pour la couverture arrière
- Les polices pré-faites de KD sont plus rapides et plus pro que du texte custom
- Choisir des couleurs de couverture qui matchent l'illustration principale

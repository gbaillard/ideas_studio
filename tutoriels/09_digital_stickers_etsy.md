# Stickers Digitaux pour Etsy (GoodNotes/Notability)

## Source YouTube
- **Titre :** How to Create Digital Stickers, Design Mockups & Sell on Etsy
- **URL :** https://www.youtube.com/watch?v=Jg1KuXPZBPw
- **Chaîne :** Design Made Clear | **Vues :** 40 071

## Process étape par étape

### Phase 1 — Créer les stickers avec Playground.com
1. Aller sur **playground.com** → catalogue de graphics commercial-use
2. Parcourir les trending templates (ex: "Literary Lovers")
3. Pour chaque sticker : modifier le texte via le prompt (ex : "change text to Reading is Cheaper Than Therapy")
4. Options : upscale + remove background (fonctionnalités pro)
5. Exporter chaque sticker en PNG
6. Répéter pour créer un set de **15 stickers**
7. Utiliser **ChatGPT** pour brainstormer des phrases thématiques

### Phase 2 — Créer les mockups pour Etsy dans Canva
1. Créer un doc Canva **1000×1000px**
2. **Mockup 1 :** Tous les stickers sur un iPad, fond bureau esthétique, titre "Digital Stickers for Book Lovers" + nombre de stickers + mention "GoodNotes"
3. **Mockup 2 :** Chaque sticker individuel sur fond blanc (pour montrer le détail)
4. **Mockup 3 :** Mention "DIGITAL DOWNLOAD ONLY - No physical items shipped" (éviter les retours)
5. **Mockup 4 :** Stickers en contexte sur un reading journal (screenshot d'un template Canva)
6. **Mockup vidéo :** Dupliquer des pages, ajouter animation "Scrapbook" → exporter en MP4

### Phase 3 — Préparer le fichier de livraison
1. Créer un dossier **Google Drive** avec tous les PNG
2. Partager le dossier : "Anyone with link" peut accéder
3. Créer un **PDF de livraison** dans Canva avec le lien Google Drive intégré dans un bouton cliquable
4. Ajouter instructions, email support, liens sociaux

### Phase 4 — Lister sur Etsy
1. **Titre SEO :** s'inspirer des concurrents — mots-clés : digital book stickers, reading, goodnotes, book lovers, notability, journals, planners, PNG
2. Pas de virgules ni de pipes dans le titre — juste des mots-clés enchaînés
3. **Description :** générée par ChatGPT avec les détails du produit
4. **Prix :** se positionner au milieu de la concurrence
5. **Quantité :** $9.99 → auto-renew activé
6. **Core details :** Digital files → AI generator (disclosure)
7. **Tags :** utiliser les 13 tags max → reprendre les mots-clés du titre
8. Uploader le PDF de livraison comme fichier digital
9. Publier

## Outils mentionnés
- **Playground.com** — génération de stickers (commercial-use, certaines fonctions gratuites)
- **Canva** — mockups et PDF de livraison
- **Google Drive** — hébergement fichiers clients
- **ChatGPT** — brainstorming phrases
- **Etsy** — marketplace

## Tips clés
- Les stickers "book lovers" sont une niche en forte croissance
- Le **mockup vidéo** augmente significativement la conversion
- Toujours ajouter un mockup "Digital Download Only" pour éviter les confusions
- Utiliser les 13 tags Etsy au maximum

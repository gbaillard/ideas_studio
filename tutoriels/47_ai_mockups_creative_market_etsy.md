## Process etape par etape

### Etape 1 : Comprendre le produit
- **Mockup de cadre** = photo realiste d'un interieur avec un cadre/poster au mur
- Les acheteurs (vendeurs d'art, designers) inserent leur propre image dans le cadre
- Vendu en JPEG/PNG (basique) ou en fichier PSD modifiable (premium)
- Niche peu concurrentielle car demande plus de travail technique
- Variations quasi infinies : styles de pieces, eclairages, angles, formats

### Etape 2 : Generation des images avec MidJourney
- Aller sur midjourney.com
- Utiliser MidJourney **version 5** (pas 5.1, trop instable pour les mockups)
- Taper `/imagine` + description
- Prompt basique : "simple poster mockup in a living room"
- Preciser le ratio : ajouter `--ar 3:2` a la fin
- Probleme courant : texte aleatoire dans le cadre + objets qui bloquent le cadre
- Solution : etre plus precis : "simple mockup of a blank poster in a minimalist living room"
- Guide de prompts gratuit fourni par l'auteur (lien description) avec 10+ exemples
- Generer differentes pieces, angles (face + profil), eclairages

### Etape 3 : Upscaling (amelioration resolution)
- Les images MidJourney sont en basse resolution = non vendables telles quelles
- Option gratuite : **Upscale.media** (45 images/mois gratuites, upscale x4)
- Option payante : **Topaz AI** (meilleure qualite)
- Importer l'image > cliquer 4x sur "Agrandir" > telecharger
- Etape indispensable pour la satisfaction client

### Etape 4a : Vente basique (JPEG/PNG)
- Vendre directement l'image agrandi au format PNG ou JPEG
- Le client insere son art via Canva, Kittle ou PhotoPea
- Vente unitaire ou par lots (3, 4, 5 mockups) ou pack complet
- Plus simple mais moins premium

### Etape 4b : Vente premium (fichier PSD modifiable dans Photoshop)
- Ouvrir l'image dans **Photoshop** (essai 7 jours gratuit) ou **PhotoPea** (gratuit)
- **Vue de face :**
  1. Outil Rectangle > dessiner un rectangle blanc dans le cadre
  2. Clic droit > Convertir en Objet Dynamique (Smart Object)
  3. Double-clic sur l'objet > inserer une image test
  4. Mode de fusion : Normal > **Produit (Multiply)**
  5. Opacite : **95%** (ajoute ombres realistes)
  6. Supprimer l'image test, masquer le rectangle, sauvegarder en .PSD
- **Vue de profil/angle :**
  1. Meme processus mais apres Smart Object : Cmd+T > clic droit > **Deformer (Warp)**
  2. Ajuster les 4 coins du rectangle aux coins du cadre incline
  3. Meme reglages Multiply + 95% opacite
  4. Sauvegarder en .PSD

### Etape 5 : Mise en vente multi-plateforme
- **Etsy** : commission 5-7%, trafic organique massif
- **Creative Market** : commission 50% (!), mais audience de designers
- **Shopify** : $32/mois, zero commission, mais necessite marketing payant
- **FreePik, Envato Elements, Storyblocks** : modele par abonnement, paiement sur base des telechargements
- Strategie ideale : vendre le MEME produit sur 3+ plateformes

## Tips cles
- **Niche peu concurrentielle** : la barriere technique (Photoshop) fait fuir la majorite des vendeurs = moins de concurrence
- **MidJourney v5 seulement** : la v5.1 est instable pour les mockups, rester sur v5
- **Toujours upscaler** : les images MidJourney brutes sont trop basses en resolution pour la vente
- **Le fichier PSD vaut plus que le JPEG** : les clients pro paient plus cher pour un template modifiable
- **Multiply + 95% opacite** : le secret pour des ombres et lumieres realistes dans le cadre
- **Multi-plateforme = revenus multiplies** : le meme produit vendu sur Etsy + Creative Market + Shopify + FreePik
- **Pas besoin de MidJourney** : on peut commencer 100% gratuit avec des photos Unsplash + PhotoPea
- **Etsy favorise la regularite** : ajouter de nouvelles annonces regulierement booste la visibilite de toute la boutique
- **La vue de profil/angle ajoute de la valeur** : les clients paient plus pour des mockups avec ombres et perspectives

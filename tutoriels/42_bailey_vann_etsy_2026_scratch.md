## Process étape par étape

### Étape 1 : Trouver un keyword rentable avec E-Rank
- Taper le **mot-clé de base** uniquement (ex: "sweatshirt", "wall art", "PNG file", "digital planner") — PAS de niche spécifique
- Analyser deux colonnes : **average monthly searches** (vert ou jaune = bon) et **competition** (vert = parfait)
- Parcourir les pages jusqu'à trouver des keywords où les deux colonnes sont vertes
- Éviter les mots-clés liés à des marques/copyrights (ex: "Wicked sweatshirt", "Wifey" = marque déposée)
- Si un keyword est jaune en volume, en empiler plusieurs similaires (ex: Grammy + Mimi + Nana + Grandma)
- Sauvegarder les keywords dans une liste pour ne pas refaire la recherche à chaque fois

### Étape 2 : Valider le style de design sur Etsy
- Taper le keyword trouvé sur Etsy (ex: "Grammy sweatshirt")
- Filtrer par **physical items** pour voir les vrais produits
- Identifier les **patterns de design** qui se vendent : grand millennial, leopard print, collegiate letters, florals, script fonts, distressed
- Utiliser **Profit Tree** pour repérer les listings < 6 mois qui vendent bien
- Objectif : comprendre quels styles visuels convertissent AVANT de designer

### Étape 3 : Créer le design avec Kittl + IA
- Aller dans les **templates Kittl** et taper un mot-clé proche (ex: "mama")
- Choisir un template tendance comme point de départ
- Créer un **Smartboard** pour modifier le template avec l'IA intégrée
- Prompt exemple : "Change the words mama to Grammy and have them be filled with a grungy leopard print pattern"
- Utiliser ChatGPT (dans Kittl) pour la meilleure adhérence au prompt
- Fond transparent activé pour export PNG
- **Cross-niching** : combiner plusieurs tendances (collegiate font + leopard print + florals)

### Étape 4 : Préparer pour Print-on-Demand (Printify)
- Convertir en image, upscaler, vérifier fond transparent
- Aller dans Printify > Catalog > choisir le produit (ex: Gildan 18000 sweatshirt)
- Uploader le design, ajuster le placement
- Sélectionner les variantes de couleur (blanc, sable = best sellers)
- Option sleeve print pour personnalisation (noms des petits-enfants = prix plus élevé)

### Étape 5 : Mockups professionnels
- NE PAS utiliser les mockups par défaut de Printify
- Acheter des **mockup bundles trendy sur Etsy** (~$10-15 pour toutes les couleurs)
- Chercher "trendy sweatshirt mockup" sur Etsy
- Adapter le mockup au buyer persona (Grammy = flatlay/folded, PAS une jeune femme avec un café)
- Baisser la transparence à 85-90% pour un rendu réaliste
- Kittl a aussi un outil mockup intégré

### Étape 6 : SEO et listing
- Copier les titres/tags des **listings performantes** qui vendent le même type de produit
- Adapter en retirant ce qui ne s'applique pas (ex: retirer "personalized" si pas customisable)
- Ajouter les spécificités de son produit (ex: "leopard print", "rose")
- Utiliser les "related searches" en bas des listings Etsy pour trouver les tags
- **Pricing** : s'aligner sur les concurrents + ajouter un discount
- **Sale 24h** : mettre un countdown permanent (renouveler chaque jour) = boost conversions confirmé

### Étape 7 : Répéter le cycle
- Refaire les 6 étapes pour chaque nouveau produit
- Lister 2-3 mois avant les saisons (ex: XOXO en janvier pour la Saint-Valentin)

## Tips clés
- **Ne jamais deviner** : chaque décision (keyword, design, style) est validée par de la data
- **E-Rank AVANT Etsy** : trouver le keyword rentable d'abord, valider le style ensuite
- **Cross-niching les tendances** : combiner leopard + collegiate + florals pour se démarquer des copies
- **Le mockup fait la vente** : un mauvais mockup tue un bon design. Investir $10-15 dans des mockups pro
- **Sale 24h permanente** : le countdown timer convertit mieux, prouvé en digital ET en POD
- **Lister en avance** : 2-3 mois avant la saison pour laisser le temps au SEO Etsy
- **Customisation = différenciation** : ajouter les prénoms sur les manches permet de facturer plus cher et réduit la concurrence
- **Un seul outil suffit** : Kittl remplace Canva + Midjourney + upscaler + mockup tool (éviter les $100/mois d'abonnements multiples)

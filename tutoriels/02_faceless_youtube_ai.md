# Chaîne YouTube Faceless avec IA

## Source YouTube
- **Titre :** How to start a faceless Youtube channel with AI [2026 FULL COURSE]
- **URL :** https://www.youtube.com/watch?v=EyXNKb2HsfM
- **Chaîne :** Syrax | **Vues :** 99 472

## Process étape par étape

### Phase 1 — Trouver une niche rentable (méthode "Incognito")
1. Ouvrir un **onglet incognito** sur YouTube
2. Chercher n'importe quelle niche faceless connue
3. Regarder 2-3 vidéos quelques minutes
4. **Rafraîchir la page d'accueil** → YouTube suggère plein de chaînes faceless
5. Chercher des chaînes **nouvelles** (< 6 mois) avec **beaucoup de vues**
6. Si une chaîne de 5 mois a 5M+ vues → la niche est bonne et pas saturée

### Phase 2 — Configurer le compte YouTube
1. Créer logo + bannière rapidement sur Canva
2. **Astuce clé :** utiliser un **compte YouTube "aged"** (2-15 ans d'ancienneté) → YouTube le trust plus → plus de reach
3. Sinon, compte neuf OK mais résultats plus lents

### Phase 3 — Trouver un sujet à fort RPM
1. Choisir des sujets qui attirent des viewers **US/UK/Canada** (pays riches)
2. Viser un public **40-60 ans** (annonceurs paient plus)
3. Utiliser ChatGPT avec ce prompt : niche + 3 vidéos virales concurrentes → génère des sujets optimisés RPM

### Phase 4 — Générer le script
1. Copier un script d'un concurrent performant (via extension NexLov)
2. Le passer dans ChatGPT avec un prompt de réécriture
3. ChatGPT génère un script dans le même style mais original

### Phase 5 — Voiceover avec ElevenLabs
1. Aller sur **ElevenLabs** → Voices → filtrer : American, Young, Male, High Quality
2. Choisir une voix → "Use voice"
3. Coller le script → Generate → Download

### Phase 6 — Créer la vidéo
1. Aller sur **TierMaker.com** → Make Template (pour niches "ranking")
2. Uploader les images (créées dans Canva ou IA)
3. Enregistrer l'écran en classant les images selon le script
4. Ouvrir **CapCut** → combiner voiceover + screen recording
5. Ajouter des GIFs de **Giphy** pour l'engagement

### Phase 7 — Thumbnail + Titre
1. Copier le layout des top chaînes de ta niche
2. **TierMaker** → screenshot du classement
3. **Canva** → YouTube thumbnail → placer screenshot + zoom + flèche
4. Garder le thumbnail **simple et clean**

## Outils mentionnés
- **ChatGPT** — sujets + scripts
- **ElevenLabs** — voiceover IA (~22€/mois)
- **TierMaker** — création de contenus "ranking"
- **CapCut** — montage vidéo (gratuit)
- **Canva** — thumbnails
- **Giphy** — GIFs d'engagement

## Tips clés du créateur
- Cas réel : 1 vidéo de 8 min = 3.8M vues = ~10 000$
- Le **sujet** détermine le revenu (pas la qualité du montage)
- RPM finance/business = 15-22$ vs entertainment = 3-5$
- **Ne pas abandonner** — les résultats viennent avec la constance

## Process étape par étape

### Phase 1 — Recherche de niches
1. Sur **Etsy**, chercher "SVG" → filtrer par "Best seller" ou "Popular now"
2. Utiliser **Etsy Marketplace Insights** (dans Shop Stats) pour voir les vrais volumes de recherche
3. Ignorer les termes IP-infringing (marques, personnages protégés)
4. Identifier les tendances qui montent (ex : SVG floraux, animaux, citations)
5. Packs de 10 à 50+ SVGs par bundle

### Phase 2 — Créer les designs avec IA
1. Utiliser un générateur IA (Midjourney, Gemini, etc.) pour créer les illustrations de base
2. Les SVGs sont des **vecteurs scalables** (ne perdent pas en qualité au zoom)
3. Différence clé vs PNG : SVG = pour machines Cricut/Silhouette, PNG = pour écrans

### Phase 3 — Vectoriser les images
1. Prendre les images IA générées (PNG)
2. Les convertir en SVG via **Vectorizer.AI** ou **SVGMaker.io**
3. Ou utiliser **Inkscape** (gratuit) / Adobe Illustrator pour nettoyer les vecteurs
4. Vérifier la qualité : zoom max, pas de pixelisation

### Phase 4 — Packager et lister
1. Créer des packs thématiques (10-50 SVGs par pack)
2. Créer des mockups montrant les SVGs sur des produits (mugs, t-shirts, découpés)
3. Lister sur Etsy avec titre SEO : "SVG Bundle [thème] for Cricut Silhouette"
4. Utiliser les 13 tags Etsy au maximum

## Qui achète des SVGs ?
- Crafters DIY avec machines Cricut/Silhouette
- Créateurs de mugs, t-shirts, stickers vinyl
- Personnes qui font du scrapbooking, des cartes
- **Ils ne veulent PAS apprendre l'IA** → ils paient pour le gain de temps

## Outils mentionnés
- **Midjourney/IA** — génération des illustrations de base
- **Vectorizer.AI / SVGMaker.io** — conversion PNG → SVG
- **Inkscape** (gratuit) — nettoyage vectoriel
- **Etsy** — marketplace

## Tips clés
- Les SVGs sont un des **produits digitaux les plus time-tested** sur Etsy
- Vendre en **packs/bundles** (pas à l'unité) → valeur perçue plus élevée
- Le marché des crafters est MASSIF et ne connaît pas l'IA
- Shops de moins de 2 ans font déjà des milliers $/mois
- L'Etsy Marketplace Insights est un outil de recherche sous-utilisé

# Vendre des Invitations de Mariage IA sur Etsy : $36K/mois

## Source YouTube
- Titre : Make $36,370 Per Month Selling AI Wedding Invites on Etsy
- URL : https://www.youtube.com/watch?v=o8wpdjp6CAA
- Chaine : Chris Heckman
- Vues : 21K
- Duree : 9 min

## Process etape par etape

### Etape 1 : Recherche du marche
- Taper "wedding invitation template" sur Etsy
- Analyser les boutiques existantes : produits digitaux (fichiers telechargeables)
- Le client telecharge, personnalise et imprime pour son mariage
- Pas de stock, pas d'expedition = profit quasi 100% (moins 5-7% frais Etsy)
- Niche etendue : baby showers, enterrements de vie de garcon/fille, remises de diplomes

### Etape 2 : Generation des designs avec MidJourney
- Aller sur midjourney.com > "Rejoindre la beta"
- Creer un serveur Discord prive + ajouter le bot MidJourney
- Taper `/imagine` puis la description du design
- Prompt simple : "minimalist wedding invitation template" suffit grace aux nouvelles versions
- MidJourney genere 4 variations
- Cliquer sur "Agrandir" (Upscale) pour la version finale
- Experimenter la barre de stylisation (50 a 1000) pour varier les rendus
- Ajouter des mots-cles : "fleurs colorees", "minimaliste", "noir et blanc"

### Etape 3 : Suppression du texte IA (post-production)
- Ouvrir l'image dans PhotoPea (outil gratuit en ligne, equivalent Photoshop)
- Utiliser l'outil "Remplissage base sur le contenu" (Content-Aware Fill)
- Selectionner tout le texte genere par l'IA (illisible)
- Appliquer le remplissage = fond uni propre
- Resultat : template vierge pret a personnaliser

### Etape 4 : Ajout du texte dans Canva
- Ouvrir Canva (gratuit)
- Importer le design nettoye
- Demander a ChatGPT les elements necessaires pour une invitation de mariage
- Ajouter les champs texte : noms des maries, date, lieu, RSVP, etc.
- Choisir des polices calligraphiques elegantes
- Adapter les couleurs au design
- Generer un lien Canva partageable pour les clients

### Etape 5 : Mise en vente sur Etsy
- Creer la fiche produit avec le lien Canva partageable
- Le client achete, recoit le lien, personnalise le texte dans Canva
- Livraison 100% automatisee par Etsy (email de suivi)
- Etsy = 450 millions de visiteurs/mois = marketing organique gratuit

### Etape 6 : Scaling
- Creer des centaines de variations (styles, couleurs, themes)
- Proposer des forfaits personnalises a prix premium
- Etendre a tous types d'evenements (pas seulement mariages)
- Creer un catalogue complet par type d'evenement

## Tips cles
- **La niche ne se limite pas aux mariages** : baby showers, graduations, enterrements de vie = demande enorme pour chaque type d'evenement
- **MidJourney ne sait pas generer du texte lisible** : toujours nettoyer le texte IA et rajouter le vrai texte dans Canva
- **Le marketing est gere par Etsy** : 450M visiteurs/mois, il suffit de creer des produits de qualite
- **La stylisation MidJourney est votre arme secrete** : varier les valeurs de 50 a 1000 pour des rendus uniques
- **Ne pas copier, mais s'inspirer** : apporter sa propre touche pour se demarquer a long terme
- **Forfaits premium** : proposer un package complet "tout le design du mariage" pour un prix bien superieur au template unitaire

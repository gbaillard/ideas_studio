## Concept
Tester si on peut déléguer la création de revenus passifs Amazon à des freelances Fiverr. Focus sur Amazon Merch on Demand (POD t-shirts).

## Comment fonctionne Amazon Merch on Demand
1. S'inscrire (gratuit, approbation nécessaire)
2. Uploader un design (image ou slogan) sur un produit (t-shirt, hoodie, popsocket)
3. Choisir les couleurs, fixer le prix
4. Amazon crée la page produit, gère le marketing, l'impression, l'expédition
5. Tu reçois un chèque mensuel avec tes royalties
6. **100% passif** après l'upload

## Autres façons de gagner de l'argent sur Amazon
- **Amazon KDP** : écrire et vendre des ebooks (royalties Kindle)
- **Amazon Mechanical Turk** : micro-tâches rémunérées
- **Amazon Merch on Demand** : POD t-shirts (focus de cette vidéo)
- **Amazon Associates** : affiliation

## Process pour Amazon Merch
1. S'inscrire sur merch.amazon.com → attendre l'approbation
2. Uploader un design (image/slogan)
3. Choisir les produits (t-shirts principalement)
4. Sélectionner les couleurs
5. Fixer le prix (markup au-dessus du coût Amazon)
6. Rédiger titre + description avec keywords
7. Amazon approuve en 1-2 jours → listing live
8. Les milliards de visiteurs Amazon trouvent ton produit via la recherche

## Avantages vs Etsy/Shopify
- **Pas de marketing** : Amazon amène les clients
- **Pas de stock** : impression à la demande
- **Pas d'expédition** : Amazon gère tout
- **Trafic gratuit** : milliards de visiteurs mensuels

## Tips clés
- Amazon Merch fonctionne par **tiers** : tu commences petit, tu unlock plus de slots avec les ventes
- Les t-shirts sont le #1 top seller sur Amazon Merch
- Combiner avec des designs IA (Canva, Nano Banana) pour du volume
- Les keywords dans le titre et la description sont cruciaux pour être trouvé

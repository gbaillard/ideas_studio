## Les 3 pratiques du Top 1%

### 1. Créer des workflows multi-outils (pas tout dans ChatGPT)
- < 10% des utilisateurs ChatGPT utilisent un 2ème chatbot
- Chaque IA a ses forces :
  - **Claude** : écriture + code
  - **Gemini** : images + vidéo + musique
  - **ChatGPT** : brainstorming + créativité
  - **Perplexity** : recherche + fact-checking
- Créer un workflow qui utilise le MEILLEUR outil pour chaque étape

### 2. Faire interviewer par l'IA AVANT de générer
- Au lieu de donner un prompt vague → demander à l'IA de POSER DES QUESTIONS d'abord
- Exemple : "Crée mon site web, mais d'abord interview-moi sur mes besoins"
- L'IA pose des questions pertinentes → collecte le contexte → produit un résultat bien meilleur
- Particulièrement utile quand on n'est PAS expert du domaine

### 3. Découper les tâches complexes en chunks
- Ne PAS demander "écris-moi un roman entier" en un prompt
- Décomposer : ending → personnages → structure → chapitres un par un
- C'est ce que font les auteurs humains (plotting avant écriture)
- Chaque chunk nourrit le suivant avec plus de contexte

## Workflow exemple : Roman avec Claude
1. Claude → idées de fins possibles
2. Claude → profils de personnages
3. Claude → structure Save the Cat (15 arcs)
4. Claude → chapitre par chapitre (avec tout le contexte précédent)

## Tips clés
- "99% of people are using AI the wrong way"
- Le multi-outils bat toujours le mono-outil
- L'interview IA = le hack le plus sous-estimé
- Le découpage en chunks = la différence entre un résultat médiocre et excellent
- Utiliser les **projets Claude** pour garder le contexte entre les sessions

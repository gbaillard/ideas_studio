# Créer des livres Shaped Word Search sur Amazon KDP (niche low competition)

## Source YouTube
- Titre : NEW Amazon KDP Niche with NO Competition - Shaped Word Search Tutorial
- URL : https://www.youtube.com/watch?v=ISmSPfWYSTw
- Chaîne : Sean Dollwet
- Vues : 115,610
- Durée : 16 min

## Process étape par étape

1. **Comprendre la niche "Shaped Word Search"**
   - Word search classique = très populaire mais compétitif
   - Shaped word search = le mot caché est dans une FORME (ours, maison, robot, trèfle, etc.)
   - Seulement 2,000 résultats sur Amazon (compétition très faible)
   - Des livres se vendent bien malgré la faible compétition

2. **Recherche de niche avec Book Bolt**
   - Aller dans la page Research de Book Bolt
   - Utiliser la fonctionnalité "Cloud" (feature favorite de Sean)
   - Trier par Top 100 dans la catégorie Activity Books
   - Garder le BSR et le price range ouvert
   - Mettre le mot-clé "word search"
   - Book Bolt affiche : couverture, éditeur, BSR, revenus estimés

3. **Créer le projet dans Book Bolt Studio**
   - Aller dans l'onglet Create > Book Bolt Studio
   - New Project > nommer le projet (ex : "Shape Word Search Puzzle")
   - Dimensions : 8.5 x 11 pouces (standard pour word search)
   - Intérieur : noir et blanc, papier blanc (le moins cher à produire)
   - Nombre de pages : 92 pages
   - Paramètre de bleed : avec bleed
   - Créer le projet

4. **Générer les puzzles shaped word search**
   - Cliquer sur Page 1 > Toolbar à gauche > bouton Template
   - Sélectionner "Word Search" parmi les options (aussi dispo : maze, sudoku, crossword, etc.)
   - Choisir un template "Shaped Word Search"
   - Configurer :
     - Board 1 : maison / Board 2 : robot / Board 3 : trèfle à 4 feuilles / Board 4 : flèche
     - Grille : 20x20 (adapté enfants) ou 30x30 (plus difficile)
     - Mots par puzzle : 10-20 (3 pour la démo)
     - Options : diagonal words, reverse words, tailles spécifiques
   - Board 5 : word search traditionnel (option invert page)
   - On peut dessiner ses propres formes (diamant, smiley, etc.)

5. **Personnaliser les mots du puzzle**
   - Créer un fichier CSV avec les mots
   - Format : mots dans la colonne A, séparés par des lignes vides entre chaque puzzle
   - Max 20 caractères par mot
   - Adapter les mots au thème de la forme (ex : maison = roof, window, door / robot = smile, metal, gears)
   - Uploader le CSV dans Book Bolt

6. **Rendre les puzzles présentables**
   - Dégrouper les éléments, repositionner le puzzle plus haut
   - Ajouter des illustrations via la File Library (Pixabay/Unsplash intégrés)
   - Filtrer par "illustrations" pour le noir et blanc
   - Ajouter des bannières, des labels pour chaque puzzle
   - Copier les éléments de design d'une page à l'autre pour gagner du temps

7. **Configurer les pages de solutions**
   - NE PAS mettre 1 solution par page (double le nombre de pages)
   - Mettre 4 solutions par page pour économiser de l'espace
   - Ajouter un border personnalisé pour chaque solution correspondant à la forme du puzzle
   - Calcul : 50 puzzles + 50 solutions = 100 pages

8. **Créer la couverture dans Book Bolt**
   - Pas besoin de template externe, tout se fait dans Book Bolt
   - Ajouter un rectangle de fond, changer la couleur
   - Ajouter le titre (ex : "Shaped Word Search")
   - Police : Archivo Black (lisible, bien pour livre enfants)
   - Effets : ombre noire subtile (angle 0, offset 1, outline 100)
   - Ajouter un aperçu de l'intérieur sur la couverture (bonne pratique)
   - Ajouter la tranche d'âge (ex : "For kids aged 3-6")
   - Ajouter les détails (ex : "80+ pages including solutions")
   - Vérifier que tout le texte est dans les marges de sécurité
   - Télécharger en CMYK (meilleur pour l'impression que RGB)

9. **Uploader sur Amazon KDP**
   - Suivre le tutoriel standard d'upload KDP

## Tips clés
- Shaped word search = niche inexploitée avec peu de compétition mais de bonnes ventes
- Utiliser 80% de l'espace des cellules pour les lettres
- Police recommandée : Open Sans pour l'intérieur
- Bordure minimum taille 5 en noir pour bien ressortir
- Ne pas gaspiller de pages : 4 solutions par page au lieu de 1
- Adapter les mots au thème de chaque forme (cohérence = meilleure expérience)
- Le format 8.5x11 noir et blanc sur papier blanc = le moins cher à produire
- On peut créer des formes personnalisées directement dans l'outil
- Penser aux variations : puzzles par état US, par animal, par saison, etc.
